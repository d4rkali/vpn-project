\# VPN Project — BS Cyber Security Semester 4



\## What this is

A working VPN built with WireGuard — a Windows desktop client (Python + Tkinter GUI)

and a Linux VPS server. Built as a semester project.



\## Stack

\- Server: Linux VPS (Hetzner) running WireGuard

\- Client: Python (CLI + Tkinter GUI) on Windows

\- Protocol: WireGuard (UDP/51820)
